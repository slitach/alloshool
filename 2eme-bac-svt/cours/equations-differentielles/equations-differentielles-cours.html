  1. Equations différentielles du premier ordre :

  1. L’équation différentielle 𝐲′=𝐚𝐲\mathbf{y}' = \mathbf{ay} (𝐚∈ℝ*)(\mathbf{a} \in \mathbb{R}^{*})

✍Activité :

On considère la fonction numérique ff définie sur ℝ\mathbb{R\ } par : f(x)=2e−4xf(x) = 2e^{- 4x}.

On pose y=f(x)y = f(x) et y′=f′(x)y' = f'(x) et y″=f″(x)y^{''} = f^{''}(x)\ .Montrer que : y″+3y′−4y=0\ y^{''} + 3y' - 4y = 0.

Toutes les équations où l’inconnue est une fonction, et qui se présente sous la forme d’une relation entre cette fonction et ses dérivées est appelée équation différentielle.

🖉🖉Propriété :

Soit aa un nombre réel non nul.

Les solutions sur ℝ\mathbb{R} de l'équation différentielle y′=ayy' = ay  est l’ensemble des fonctions définies sur ℝ\mathbb{R}  par : f(x)=keaxf(x) = ke^{ax}\ où kk est un nombre réel quelconque.

🔿Exemple :

L’ensemble des solutions de l’équation différentielle (E):y′+2y=0(E):y' + 2y = 0 est l’ensemble des fonctions définies sur ℝ\mathbb{R}  par : f(x)=ke−2xf(x) = ke^{- 2x}\ avec k∈ℝk\mathbb{\in R}.

🖉🖉Propriété :

L’ensemble des solutions de l’équation différentielle y′=ay+by' = ay + b est l’ensemble des fonctions ff définies sur ℝ\mathbb{R} par : f(x)=keax−ba(x) = ke^{ax} - \frac{b}{a}.

🔿Exemple :

L’ensemble des solutions de l’équation différentielle (E):y′=3y+2(E):y' = 3y + 2 est l’ensemble des fonctions définies sur ℝ\mathbb{R}  par :f(x)=ke3x−23f(x) = ke^{3x} - \frac{2}{3}\ avec k∈ℝk\mathbb{\in R}.

Déterminons la solution gg de l’équation (E)(E) qui vérifie la condition g(−1)=13g( - 1) = \frac{1}{3}.

Pour tout x∈ℝ,g(x)=ke3x−23x\mathbb{\in R\ ,\ }g(x) = ke^{3x} - \frac{2}{3}\ \ .

La condition g(−1)=13g( - 1) = \frac{1}{3} donne ke−3−23=13ke^{- 3} - \frac{2}{3} = \frac{1}{3}.

Donc k=e3k = e^{3} il s’ensuit donc : (∀x∈ℝ):g(x)=e3x+3−23\left( \forall x\mathbb{\in R} \right)\ :g(x) = e^{3x + 3} - \frac{2}{3}.

  1. Equation différentielles du second ordre : y″+ay′+by=0y^{''} + ay' + by = 0

🖉🖉 Définition :

Soient aetba\ et\ b deux nombres réels.

L’équation r2+ar+b=0r^{2} + ar + b = 0 , où rr est l’inconnue, s’appelle l’équation caractéristique de l’équation différentielle y″+ay′+by=0y^{''} + ay' + by = 0.

🖉🖉Propriété :

On considère (E)(E) l’équation différentielle (E):y″+ay′+by=0(E):\ y^{''} + ay' + by = 0 et soit Δ\Delta le discriminant de son équation caractéristique (E′):r2+ar+b=0(E'):\ r^{2} + ar + b = 0.

  • Si Δ>0\Delta > 0, alors les solutions de (E)(E) sont les fonctions x↦αer1x+βer2xx \mapsto \alpha e^{r_{1}x} + \beta e^{r_{2}x} tel que (α,β)∈ℝ2(\alpha,\beta) \in \mathbb{R}^{2} où r1r_{1} et r2\ r_{2} sont les solutions de (E′)(E').

  • Si Δ=0\Delta = 0, alors les solutions de (E)(E) sont les fonctions x↦(α+βx)er0xx \mapsto (\alpha + \beta x)e^{r_{0}x} tel que (α,β)∈ℝ2(\alpha,\beta) \in \mathbb{R}^{2} où r0r_{0} est la solution de (E′)(E').

  • Si Δ<0\Delta < 0, alors les solutions de (E)(E) sont les fonctions x↦(αcos(qx)+βsin(qx))epxx \mapsto \left( \alpha\cos(qx) + \beta\sin(qx) \right)e^{px} tel que (α,β)∈ℝ2(\alpha,\beta) \in \mathbb{R}^{2} où r1=p+iqr_{1} = p + iq et r2=r1¯\ r_{2} = \overline{r_{1}}sont les solutions de (E′)(E').
