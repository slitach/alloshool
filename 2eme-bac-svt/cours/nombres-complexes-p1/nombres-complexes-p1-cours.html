  1. L’ensemble des nombres complexes 

  1. L’ensemble ℂ\mathbb{C} - L’écriture algébrique

🖉🖉Théorème :

Il existe un ensemble noté ℂ\mathbb{C} appelé l’ensemble des nombres complexes, il contient l’ensemble ℝ\mathbb{R} et il vérifie ce qui suit :

  • L’ensemble ℂ\mathbb{C} contient un élément irréel ii qui vérifie i2=−1i^{2} = - 1.

  • Tout élément zz de ℂ\mathbb{C} s’écrit de façon unique sous la forme z=a+ibz = a + ib où aetba\ et\ b\ sont deux nombres réels.

🔿 Vocabulaires :

  • L’écriture z=a+ibz = a + ib s’appelle l’écriture algébrique du nombre complexe zz\ .

  • Le nombre aa est appelé partie réelle du nombre zz qu’on note par ℛℯ(z)\mathcal{Re}(z).

  • Le nombre bb est appelé partie imaginaire du nombre zz qu’on note ℐ𝓂(z)\mathcal{Im}(z).

  • Tout nombre qui s’écrit sous la forme ibib est dit nombre imaginaire pur, et l’ensemble des nombres imaginaires purs est noté iℝi\mathbb{R}.

🔿Exemples :

  • 𝐳𝟏=5+7i\mathbf{z}_{\mathbf{1}} = 5 + 7i, alorsℛℯ(z1)=2etℐ𝓂(z1)=7\mathcal{\ Re}\left( z_{1} \right) = 2\ et\ \mathcal{Im}\left( z_{1} \right) = 7.

  • 𝐳𝟐=1−i\mathbf{z}_{\mathbf{2}} = 1 - i, alorsℛℯ(z2)=1etℐ𝓂(z2)=−1\mathcal{\ Re}\left( z_{2} \right) = 1\ et\ \mathcal{Im}\left( z_{2} \right) = - 1.

  • 𝐳𝟑=5,\mathbf{z}_{\mathbf{3}} = 5\ ,\ alors ℛℯ(z3)=5etℐ𝓂(z3)=0\mathcal{\ Re}\left( z_{3} \right) = 5\ et\ \mathcal{Im}\left( z_{3} \right) = 0.

  • 𝐳𝟒=3i\mathbf{z}_{\mathbf{4}} = 3i, alors ℛℯ(z4)=0etℐ𝓂(z4)=3\mathcal{\ Re}\left( z_{4} \right) = 0\ et\ \mathcal{Im}\left( z_{4} \right) = 3 (on a z4z_{4} est un nombre imaginaire pur)

  • 𝐳𝟓=1−5+2−π2+4i,\mathbf{z}_{\mathbf{5}} = 1 - \sqrt{5} + \sqrt{2} - \frac{\pi}{2} + 4i,\ alorsℛℯ(z5)=1−5+2−π2etℐ𝓂(z5)=4\mathcal{\ Re}\left( z_{5} \right) = 1 - \sqrt{5} + \sqrt{2} - \frac{\pi}{2}\ et\ \mathcal{Im}\left( z_{5} \right) = 4.

Soientz=a+ib\ z = a + ib et z′=a′+ib′\ z' = a' + ib' deux nombres complexes on a :

  • z=z′⇔a=a′etb=b′z = z' \Longleftrightarrow a = a'et\ b = b'.

  • z=0⇔a=0etb=0z = 0 \Longleftrightarrow a = 0\ et\ b = 0.

  1. Opérations dans ℂ\mathbb{C}

Les opérations de la somme et le produit de ℝ\mathbb{R} se prolongent en ℂ\mathbb{C} et elles ont les mêmes propriétés.

🖉🖉Propriété :

  • 1z=1a+ib=a−ib(a+ib)×(a−ib)=a−iba²+b²(z≠0)\frac{1}{z} = \frac{1}{a + ib} = \frac{a - ib}{(a + ib) \times (a - ib)} = \frac{a - ib}{a² + b²}\ (z \neq 0)

  1. Colinéarité de deux points 

🖉🖉Propriété :

Soient A(zA)A\left( z_{A} \right),B(zB)B\left( z_{B} \right)et C(zC)C\left( z_{C} \right) trois points du plan complexe.

Les points AA,BB et CC sont alignés si et seulement si zB−zAzC−zA∈ℝ\frac{z_{B} - z_{A}}{z_{C} - z_{A}}\mathbb{\in R}.

🔿 Démonstration :

Les points AA,BB et CC sont alignés si et seulement il existe un réel kk tel que AB→=kAC→\overrightarrow{AB} = k\overrightarrow{AC}.

Et on a : AB→=kAC→⇔zB−zA=k(zC−zA)⇔zB−zAzC−zA=k∈ℝ\overrightarrow{AB} = k\overrightarrow{AC} \Longleftrightarrow z_{B} - z_{A} = k\left( z_{C} - z_{A} \right) \Longleftrightarrow \frac{z_{B} - z_{A}}{z_{C} - z_{A}} = k\mathbb{\in R}.

✍ Application ⑥:

  1. Conjugué d’un nombre complexe

🖉🖉 Définition :

Soit z=a+ibz = a + ib, où aa et bb sont deux réels.

Le nombre a−iba - ib est appelé le conjugué du nombre complexe zz, et on le note par z¯\overline{z}.

🔿Exemples :

  • z3=i{\ z}_{3} = i, alors z3¯=−i\overline{{\ z}_{3}} = - i.

  • z4=6{\ z}_{4} = 6, alors z4¯=6\overline{{\ z}_{4}} = 6.

Soit zz un nombre complexe.

  • z×z′¯=z¯×z′¯\overline{z \times z'} = \overline{z} \times \overline{z'}\ ;
